# Varnish Router Grpc Structure

## 1. Introduction

Varnish Router gRPC is a gRPC client for Varnish Router plugin creation.

## 2. Installation

### 2.1. Requirements

    * GO 1.13 or higher
    * Protoc 3.11.4 or higher
    * Protoc-gen-go 1.3.2 or higher
    * Varnish console
    * Varnish Router plugin
